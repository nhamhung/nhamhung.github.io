# Mentoring for the SIM-LSE Data Analytics Challenge 2026

This sample post lives directly inside the portfolio repository. Students can duplicate this Markdown file, change the metadata in `src/data/journalPosts.ts`, and publish their own learning notes without setting up WordPress or another blog.

## What I Learned

- A portfolio can be both a showcase and a learning journal.
- Static Markdown content works well for GitHub Pages.
- Writing short reflections helps make technical growth visible.

## How To Use This

Replace this content with your own notes, project retrospectives, study plans, or teaching reflections. Keep the post focused on one learning moment so readers can quickly understand what changed in your thinking.
