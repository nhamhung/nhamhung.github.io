# Build Instructions

## Prerequisites

| Requirement | Value |
|---|---|
| Runtime | Node.js 20 or newer |
| Package manager | npm |
| Build tool | Vite |
| Language | TypeScript |
| Deployment output | `dist/` |
| Required secrets | None for local build |

## Build Steps

### 1. Install Dependencies

```bash
npm install
```

Use this after cloning the repository or after dependency changes.

### 2. Run Automated Tests

```bash
npm run test
```

Expected result:

- Vitest completes successfully.
- Current Unit 5 baseline: 3 test files and 9 tests pass.

### 3. Build The Static Site

```bash
npm run build
```

Expected result:

- TypeScript compiles successfully.
- Vite creates production output in `dist/`.

Acceptable current warning:

- Vite may warn that some chunks are larger than 500 kB because the portfolio includes large images and PDF assets. This warning does not fail the build.

### 4. Preview The Production Build

```bash
npm run preview
```

Expected result:

- Vite serves the production build locally.
- Open the printed local URL and check major sections before publishing.

## Build Artifacts

| Artifact | Description |
|---|---|
| `dist/` | Static production site generated by Vite. |
| `dist/index.html` | Built app entry point. |
| `dist/assets/` | Bundled JS, CSS, images, and PDFs. |

## Troubleshooting

### Dependency Install Fails

Likely causes:

- Node.js version is older than 20.
- npm cache or network issue.
- `package-lock.json` conflicts after manual edits.

Try:

```bash
node --version
npm --version
npm install
```

### Build Fails With TypeScript Errors

Likely causes:

- A `src/data/` file no longer matches `src/types/portfolio.ts`.
- An import path points to a missing asset.
- A test or config type is invalid.

Try:

```bash
npm run test
npm run build
```

Then inspect the first TypeScript error and fix the named file.

### Assets Missing In Build

Likely causes:

- A file in `src/assets/` was renamed or deleted.
- A matching import in `src/data/` was not updated.

Fix the import or restore the asset, then rerun:

```bash
npm run build
```
